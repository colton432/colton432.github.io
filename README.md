# Budget Buddy

Budget Buddy is a Flutter budgeting app for tracking personal spending locally on a device. It is designed for users who want a simple, private way to manage transactions, subscriptions, spending trends, and budget summaries without connecting to a bank account or cloud database.

## About the App

Budget Buddy lets users add, edit, search, filter, and delete transactions. Each transaction can include a name, amount, date, notes, and category. The app includes default categories such as Gas, Food, Rent, Entertainment, Shopping, and Other, while also supporting custom categories.

The app also includes recurring subscription tracking. Users can manage monthly or yearly subscriptions with active or inactive status, billing dates, categories, notes, upcoming bills, and estimated monthly or yearly subscription totals.

## Main Features

- Add, edit, search, filter, and delete transactions
- Track transaction names, amounts, dates, notes, and categories
- Use default categories such as Gas, Food, Rent, Entertainment, Shopping, and Other
- Create and manage custom categories
- Track recurring subscriptions
- Support monthly and yearly subscription billing
- Mark subscriptions as active or inactive
- View upcoming subscription bills
- Estimate monthly and yearly subscription costs
- View current month spending summaries
- See total spending, recurring costs, top category, category cards, upcoming subscriptions, and recent transactions
- Browse category-specific spending views
- Review transaction history
- View monthly trend charts
- View pie and line spending visualizations
- Generate yearly summaries
- Review month-by-month and category-by-category breakdowns
- Export budget summaries as PDF previews or CSV files
- Export data for a selected month, full year, or custom date range
- Use light mode or dark mode
- Mobile banner ad support for Android and iOS

## Privacy and Data Storage

Budget Buddy is private by default. The app stores user data locally on the device using `shared_preferences`.

Budget Buddy does not use a cloud backend, does not require user accounts, and does not connect to bank accounts.

Because the data is stored locally, deleting the app may remove saved Budget Buddy data from the device unless the user has exported their information.
